# FUNDEB Fácil

Sistema Inteligente para Compreensão e Projeção de Complementações Orçamentárias do FUNDEB

## Prêmio SOF 2025
Categoria: Soluções em dados orçamentários
Desafio: Item 2.4.1.4 - Estimar ou visualizar projeções orçamentárias de médio prazo

## Descrição

O FUNDEB Fácil é uma aplicação web inteligente que utiliza arquitetura de multiagentes para transformar a complexidade regulatória do FUNDEB em compreensão acionável para gestores municipais.

## Funcionalidades

- Calculadora interativa de complementações VAAT e VAAF
- Assistente conversacional com IA para explicações pedagógicas
- Projeções orçamentárias baseadas em cenários de matrículas
- Visualizações interativas e intuitivas

## Tecnologias

- Streamlit
- LangGraph (multiagentes)
- Claude 3.5 Sonnet (Anthropic)
- SQLite
- Plotly

## Como executar

```bash
uv venv
uv pip install -e .
streamlit run app.py
```
